
# Vigenère cipher solver

Uses index of coincidence and frequency analysis to find Vigenère cipher keys
from ciphertexts.

## Build

```bash
make
```

Binary will be written to `./build/solve`

## Example usage

```
./build/solve txt/shakespeare-encrypted.txt
```
